## 数组 & 链表
**数组**
连续的内存空间，空间大小固定，支持随机访问，插入删除操作效率低。

**链表**
不是连续的内存空间，空间大小不固定，支持顺序访问，插入删除操作效率高。
链表有两种：
* 单链表
* 双链表
